# TubeIQ — YouTube Intelligence Platform
Analyze any YouTube channel instantly.
Predict video performance, discover topic clusters,
compare competitors and find content gaps.

## Features
- Any channel analysis in seconds
- XGBoost video performance predictor
- K-Means topic clustering
- Competitor channel comparison
- Best upload time recommendations
- Content gap finder

## Stack
YouTube Data API v3 → FastAPI → PostgreSQL →
XGBoost + K-Means → React + Recharts + Tailwind
